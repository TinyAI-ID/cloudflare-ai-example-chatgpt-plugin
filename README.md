# ChatGPT Plugins on Cloudflare

This repo contains Cloudflare's work with ChatGPT plugins. The plugins are built using the [Cloudflare Workers](https://workers.cloudflare.com/) platform.

- [`example-plugin`](https://github.com/cloudflare/chatgpt-plugin): An example plugin that can be deployed to Cloudflare Workers in 5 minutes, allowing search and retrieval of GitHub repositories.
